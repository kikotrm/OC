Perguntas Frequentes (FAQ)

Encontra respostas para as dúvidas mais comuns abaixo.

1. Como posso alterar a minha palavra-passe? +
  1. Acede à tua página de perfil.
  2. Clica no botão "Recuperar Palavra-passe".
  3. Receberás um email com instruções para redefinir a tua palavra-passe.
  4. Segue as instruções no email para concluir o processo.
2. Como edito o meu nome de utilizador? +
  1. Acede à tua página de perfil.
  2. Clica no botão "Editar Perfil".
  3. Altera o nome de utilizador no campo indicado.
  4. Clica em "Guardar" para confirmar as alterações.
3. Como posso contactar o suporte? +
Para entrares em contacto com o suporte, acede à secção "Contactar-nos" no menu superior e preenche o formulário. A nossa equipa irá responder-te o mais brevemente possível.
4. Os meus dados estão seguros na OpenCourt? +
Sim! Levamos a tua segurança muito a sério. Utilizamos autenticação segura via Firebase e armazenamento encriptado para proteger os teus dados pessoais.
